## CH05 开发你的第一个 MCP（上）

### 为什么要学习 MCP 开发

只有深入了解 MCP 工具，才能在选择 MCP 工具时更明智的做判断。

如果当前的 MCP 工具不能满足个人的需求，那么能自行动手开发一个 MCP 是个更好的选择。

从使用者的身份转为创作者，能够更好地理解 MCP 及其生态的现状和发展潜力。

---

## 三大核心组件

### 1、Tool（工具）

**作用**：可调用的函数

**功能**：决定了 MCP 可以做什么

---

### 2、Resource（资源）

**作用**：可读取的数据

**功能**：决定了 MCP 可以有什么

---

### 3、Prompt（提示模板）

**作用**：可复用的提示词

**功能**：决定了 MCP 的复用能力

---
